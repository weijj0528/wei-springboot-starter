## 快速开始

### 接口文档地址

[接口文档](http://localhost:8080/doc.html)

#### 注意：线上环境关闭接口文档

```yaml

# Swagger
swagger:
    enabled: false

```

### 代码生成

本项目使用的代码生成工具是 [EasyCode Idea 插件](https://plugins.jetbrains.com/plugin/10954-easycode)

导入项目根目录下 [EasyCodeConfig.json](EasyCodeConfig.json) 配置文件

按照插件提示操作即可
